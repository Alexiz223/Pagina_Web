🎨 Inspiración
El diseño está inspirado en tendencias actuales vistas en plataformas como:

Landingfolio
Dribbble

Se buscó replicar:

Tipografía bold y moderna
Uso de colores vibrantes (menta, lavanda, naranja)
Layout tipo agencia creativa
Componentes visuales llamativos


Tecnologías utilizadas

Python 3.11+
Reflex 0.6+
Node.js 18+

Instalación
1. Clonar el repositorio

git clone https://github.com/Alexiz223/Pagina_Web.git

2. Entrar al proyecto

cd Pagina_Web

3. Crear entorno virtual

python -m venv venv

4. Activar entorno virtual

venv\Scripts\activate

5. Instalar dependencias

bashpip install -r requirements.txt

6. Inicializar Reflex

reflex init

Ejecutar el proyecto

reflex run

Luego abrir en el navegador:
http://localhost:3000
